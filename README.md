# 👨‍💼 Employee Management System

A simple Employee Management System built with Python that manages employee records using JSON storage. The project demonstrates CRUD operations and modular programming.

## Features

- Add employee
- View employee records
- Update employee information
- Delete employee
- JSON data persistence
- Input validation

## Technologies Used

- Python 3
- JSON
- Git
- GitHub
